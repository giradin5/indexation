Sitemap de Mon Site

  • Étape 1 : Accéder à XML Sitemaps

    ------Ouvre ton navigateur internet et va sur XML Sitemaps.

  • Étape 2 : Entrez l'URL de votre site

    -------Trouve le champ "Starting URL", entre l'adresse de ton site web et clique sur "Start".

  • Étape 3 : Attendre la génération du sitemap

    Patiente quelques minutes pendant que XML Sitemaps analyse ton site. Regarde la progression affichée.

  • Étape 4 : Télécharge le sitemap

    ------Une fois terminé, clique sur "Download your XML file" et sauvegarde le fichier sur ton ordinateur.

  • Étape 5 : Ouvrir votre fichier HTML avec Bootstrap

    -------Ouvre un éditeur de texte (comme Notepad ou VSCode), crée un nouveau fichier et nomme-le "sitemap.html".

  • Étape 6 : Ajouter la structure de base de Bootstrap

    -------Utilise le code HTML de base avec les styles Bootstrap et personnalisés.

  • Étape 7 : Ajouter les liens du sitemap
  • Étape 8 : Enregistrer et visualiser

    -------Enregistre le fichier HTML et ouvre-le dans un navigateur.

  • Étape 9 : Vérifier le sitemap

    -------Vérifie que tous les liens s'affichent correctement et clique sur les liens pour s'assurer qu'ils fonctionnent.

  • Étape 10 : Déployer sur votre site

    -------Téléverse le fichier "sitemap.html" sur ton serveur web et place le fichier à la racine de ton site web.
